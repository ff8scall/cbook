# 체크리스트: 변동 URL 감지 및 동적 스케줄링 고도화 (2026-05-02)

## 📌 개요
리디북스 등 플랫폼의 이벤트 페이지 URL이 수시로 변경됨에 따라, 하드코딩된 URL 대신 동적으로 현재 활성화된 이벤트 URL을 찾아내거나 변동을 감지하는 메커니즘을 도입합니다.

## 👨‍💻 MASC 분석 결과
- **PM (기획)**: "매번 URL을 수동으로 따오는 건 비효율적입니다. '현재 진행 중인 대규모 할인'을 자동으로 찾는 기능이 핵심입니다."
- **Architect (설계)**: "이벤트 목록 페이지를 먼저 크롤링하여 키워드(예: '심쿵', '50%', '세트 할인')가 포함된 최신 링크를 추출하는 'URL Discovery' 레이어를 추가하겠습니다."
- **Security (보안)**: "탐색 빈도를 조절하여 IP 차단을 방지하고, 비정상적인 리다이렉션 발생 시 알림을 보내는 예외 처리가 필요합니다."
