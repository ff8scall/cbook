# 체크리스트: 누락 이미지 복구 및 알라딘 API 데이터 통합 고도화 (2026-05-02)

## 📌 개요
현재 대시보드에 이미지가 누락된 도서가 다수 발견됨에 따라, 이미지 추출 로직을 강화하고 알라딘 API의 데이터를 활용한 2중 방어 체계(Image Fallback)를 구축합니다.

## 👨‍💻 MASC 분석 결과
- **PM (기획)**: "이미지는 구매 결정의 핵심 요소입니다. 썸네일 누락률을 5% 미만으로 낮추는 것이 목표입니다."
- **Architect (설계)**: "알라딘 API가 이미 정가 조회를 위해 호출되고 있으므로, 응답에 포함된 `cover` 이미지 URL을 함께 반환받아 활용하는 것이 효율적입니다."
- **Security (보안)**: "성인물 도서의 경우 알라딘 API에서 가져온 이미지도 마스킹 처리되어야 함을 잊지 마세요."
