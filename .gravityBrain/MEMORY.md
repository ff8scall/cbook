## ✅ 최근 결정 사항
- **예스24(YES24) 플랫폼 추가**: 만화 세트 할인 수집 대상을 4개 플랫폼으로 확장 완료.
- **스크레이퍼 구조 개편**: 플랫폼별 독립 실행 옵션(`npm run scrape:yes24` 등) 도입으로 데이터 업데이트 유연성 확보.
- **성인물 보호 로직 적용**: 19금 상품 감지 시 원본 표지 대신 전용 Placeholder 이미지(`adult_placeholder.png`) 노출 강제.
- **교보문고 필터링 강화**: '판매금지/품절/종료' 상품을 수집 단계에서 원천 차단하여 허위 정보 제거.
- **데이터 저장 방식 최적화**: 수집 시마다 기존 데이터를 초기화(Overwrite)하고 플랫폼별 파일을 분리하여 UI 동기화 속도 개선.
