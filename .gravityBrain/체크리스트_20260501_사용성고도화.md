# 체크리스트: 대시보드 사용성 고도화 및 검색 강화 (20260501)

## 1. 개요
대시보드의 직관성을 높이기 위해 썸네일 이미지에 링크 기능을 부여하고, 검색 기능을 제목에서 저자까지 확장하며, 성인물 보호 로직을 상시 적용합니다.

## 2. 상세 작업 단계
- [x] **[UI] 썸네일 클릭 시 상세 링크 연결**
    - [x] `app.js`의 `renderDeals` 수정: `displayOffer` 정보를 바탕으로 썸네일을 `<a>` 태그로 감싸기.
    - [x] 업체가 1개인 경우 뿐만 아니라, 기본 노출되는 '최저가 업체'로 바로 연결되도록 범용적으로 구현.
- [x] **[Search] 제목 및 작가 통합 검색 구현**
    - [x] `app.js`의 `searchQuery` 필터 로직 수정: `title`뿐만 아니라 `author` 필드도 검색 대상에 포함.
- [x] **[Filter] 성인물 기본 숨김 로직 상시화**
    - [x] 페이지 로드 시 `showAdult = false`가 확실히 적용되는지 재확인.
    - [x] 19금 콘텐츠의 제목이나 작가도 필터링 상태에 따라 안전하게 처리되는지 검토.
- [x] **[Logic] 정밀 검수**
    - [x] 검색 시 대소문자 구분 없이 제목/작가 모두에서 결과가 나오는지 테스트.
    - [x] 이미지 클릭 시 새 탭(`_blank`)으로 열리는지 확인.
    - [x] **성인물 필터링 강화**: '용주골', '대물' 등 키워드 기반 차단 로직 추가 검증 완료.

## 3. 리스크 및 고려사항
- **UX 일관성**: 이미지 클릭 시 이동과 하단 플랫폼 버튼 클릭 시 이동이 중복될 수 있으나, 최저가 바로가기라는 측면에서 긍정적임.
- **성인물 판별**: 작가 이름에 성인 키워드가 포함된 경우(드문 케이스) 오판별 가능성이 있으므로 기존 판별 로직 유지.
